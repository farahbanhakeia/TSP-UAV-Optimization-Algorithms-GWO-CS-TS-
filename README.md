Ce dépôt contient une implémentation et une visualisation du problème TSP appliqué à un UAV dans le contexte de l’Internet des Objets (IoT).

Plusieurs algorithmes métaheuristiques sont utilisés et comparés :
*Grey Wolf Optimizer (GWO)
*Cuckoo Search (CS)
*Tabu Search (TS)
*Génération aléatoire de trajectoires

-Le projet permet :
*La visualisation dynamique des trajectoires UAV
*L’analyse des performances des algorithmes
*L’export des trajectoires dans des fichiers CSV
*Langage principal : Python
